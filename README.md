Neural Tool Kit (NTK) is a python machine learning framework, built from scratch. This project aims to serve as useful machine learning utility, as well as an educational endevour. Through developing Neural Tool Kit, my goal is to build a strong understanding of the systems behind modern machine learning techniques, as well as fostering in myself a strong suite of technical skills that can be applied in numerous fields. Neural Tool Kit is under active development. It provides a streamlined process to quickly build, train, and tests models, as well as tensor abstraction and auto gradient differentiation.
